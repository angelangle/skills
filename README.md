# Skills Repository

Personal collection of Claude Code agent skills.

## Skills

### skill-refiner
Evolve and optimize agent skills through systematic diagnosis, analysis, and improvement.

**When to use:** Improving skill capabilities, optimizing triggers, diagnosing skill issues, analyzing skill gaps, merging skills, or discovering similar skills.

**Features:**
- Skill discovery and local/external similarity detection
- Skill health analysis and diagnostics
- Skill merging and fusion capabilities
- Reference management and engineering practices
- Workflow optimization tools

**Location:** `skill-refiner/`

## Usage

These skills are designed for use with [Claude Code](https://claude.ai/code).

To use a skill, reference it in your conversation with Claude or use the Skill tool directly.
